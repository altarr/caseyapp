# Keyboard Shortcuts & Accessibility

## Goal
Add keyboard shortcuts and accessibility improvements to all presenter pages.

## Success Criteria
1. presenter/lib/shortcuts.js created with ESC/S/A/D/F/N/P/? shortcuts
2. Help overlay shows all shortcuts when ? is pressed
3. All presenter HTML files include shortcuts.js
4. ARIA labels on all interactive elements
5. Proper focus management and screen reader friendly content
6. Shortcuts don't fire when typing in inputs/textareas
