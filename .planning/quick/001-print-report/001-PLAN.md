# Print Report Feature

## Goal
Add a "Print Report" button to session-viewer.html that opens a clean, white-background printer-friendly version of the session analysis report.

## Success Criteria
1. Print Report button visible in the session viewer topbar
2. Clicking opens a print-optimized view with white background
3. Includes: company logo placeholder, visitor info, executive summary, key interests table, follow-up actions checklist, key moments with timestamps
4. Uses @media print CSS for proper print rendering
5. No dark backgrounds, no scrollable containers in print view
6. Clean typography suitable for paper output
