# Demo Landing Page

## Goal
Create a demo landing page at `demo/index.html` that displays on the demo PC during booth demos, showing real-time session status by polling S3 for `active-session.json`.

## Success Criteria
1. Header shows "BoothApp - AI Demo Capture" with team name "Smells Like Machine Learning"
2. Displays current session status (active/idle) by polling S3 for active-session.json
3. Shows visitor name and session duration when active
4. Shows a live click counter
5. Dark theme, modern fonts (Google Fonts), responsive
6. Configurable S3 endpoint at top of JS
7. PR to main
