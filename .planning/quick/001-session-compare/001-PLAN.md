# Session Comparison View

## Goal
Create presenter/compare.html that lets hackathon judges compare two demo sessions side-by-side.

## Success Criteria
1. URL params ?a=<id>&b=<id> load both sessions
2. Side-by-side visitor info cards (name, company, SE, status)
3. Duration comparison bar chart (pure CSS/canvas)
4. Click count comparison
5. Products demonstrated comparison (overlap list showing shared vs unique)
6. Analysis score comparison (engagement, coverage, follow-up)
7. Transcript word cloud for each session
8. Consistent dark theme matching existing presenter pages
9. Navigation links to/from sessions list
