# Plan: Beautiful Dark Report Template

## Goal
Redesign the HTML report template (templates/report.html) with a dark gradient theme, modern typography, and visitor-facing presentation suitable for emailing to trade show visitors.

## Success Criteria
1. Dark gradient theme throughout (not light/white background)
2. Branded header with BoothApp/Trend Vision One logo text
3. Visitor name and company prominently displayed
4. Session date and duration visible
5. Executive summary section
6. Key products demonstrated with bullet points
7. Engagement highlights section
8. Personalized follow-up recommendations
9. Professional footer
10. Modern typography (system font stack + monospace for data)
11. Responsive layout (mobile-friendly)
12. Self-contained single HTML file (all CSS inline, no external deps)
13. Existing render-report.js placeholders still work (no JS changes needed)
