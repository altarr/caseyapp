# ROI Calculator for Presenter Page

## Goal
Add an ROI calculator page to the presenter dashboard that shows estimated XDR ROI based on team size input, using industry benchmarks for investigation time, MTTR, and breach prevention.

## Success Criteria
1. New `roi-calculator.html` page in presenter/ with dark theme matching existing pages
2. Team size input (slider or number input) that dynamically recalculates all values
3. Three ROI categories displayed as infographic cards:
   - Time saved per investigation: 45min manual vs 5min XDR
   - MTTR reduction: 287 hours vs 24 hours
   - Breach prevention from automated playbooks
4. Dollar amounts calculated from team size and industry benchmarks
5. Clean infographic layout with animated counters
6. Added to presenter nav component
7. Accessible from quick-nav links on dashboard
